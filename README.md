DOKI DOKI
=============
[Mario-Level-1](https://github.com/justinmeister/Mario-Level-1)을 이용해 만든 게임입니다.

<img src ='https://github.com/kohyerim/DOKIDOKI/blob/master/screenshot.png'></img>


### 조작
좌 우 방향키를 이용해 이동합니다!
스페이스바를 이용해 점프합니다!



### 다운받기 전에
- Python 3과 Pygame이 설치되어 있어야 합니다.
- Pygame 설치
        pip install pygame


### 구현 전
- 스테이지 별 캐릭터 이미지 변경
- 스테이지(레벨) 디자인
